## Project Overview

AppleCrafts is an e-commerce platform built using the MERN stack, designed to provide a seamless shopping experience for customers looking for high-quality iPhones. The application features a comprehensive set of functionalities, including payment processing, order management, and a digital wallet system.

## Features

- **Product Showcase**: Display of the latest iPhone models with detailed descriptions and high-quality images.
- **Responsive Design**: Optimized for both desktop and mobile devices using Tailwind CSS.
- **User Authentication**: Secure login and registration system for customers, including role-based authentication for admins and users.
- **Shopping Cart**: Easily manage your selected products before checkout.
- **Payment Integration**: Secure payment processing with Razorpay.
- **Order Management**: Comprehensive order tracking and management system.
- **Digital Wallet**: Wallet feature for storing credits and making purchases.
- **Admin Dashboard**: Advanced management tools for admins to handle products, orders, and users.
- **Email Notifications**: Automated email notifications for orders and other important events using Nodemailer.
- **Charts & Analytics**: Visualize sales data and other metrics using Chart.js.

## Technologies Used

- **Frontend**: React, Redux, Tailwind CSS
- **Backend**: Node.js, Express.js
- **Database**: MongoDB
- **Authentication**: OAuth 2.0, JSON Web Tokens (JWT)
- **Payment Gateway**: Razorpay
- **File Uploads**: Multer
- **Email Service**: Nodemailer
- **Hosting & Deployment**: AWS EC2, Nginx
- **Version Control**: Git
